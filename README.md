> A curated list of tools, frameworks, libraries, and resources for analyzing flight logs from drones (UAVs), aircraft, and other aerial vehicles.

Flight log analysis is essential for understanding flight performance, diagnosing issues, and improving safety in aviation and drone operations. This list is aimed at drone pilots, UAV developers, aviation engineers, and enthusiasts.

## Open-Source Tools

- [Flight Review](https://github.com/PX4/flight_review) – Web-based flight log analysis and visualization for PX4 autopilot logs.
- [DroneLogbook Tools](https://github.com/DroneLogbook) – Tools and utilities for managing and analyzing drone logs.
- [Mission Planner](https://github.com/ArduPilot/MissionPlanner) – Ground control station software with flight log analysis features.
- [APM Log Analyzer](https://github.com/ArduPilot/apm_log_analyzer) – Python tool for analyzing ArduPilot log files.

## Frameworks & Libraries

- [PyFlightAnalysis](https://github.com/ArduPilot/pyflightanalysis) – Python library for parsing and analyzing flight logs from ArduPilot.
- [MAVExplorer](https://github.com/ArduPilot/MAVExplorer) – Log analysis tool in Python for ArduPilot log files with graphing and scripting.
- [Ulog Analysis Tools](https://github.com/PX4/ulog) – Python scripts for analyzing PX4 ULog files.

## Commercial Tools

- [Airdata UAV](https://airdata.com/) – Cloud-based flight log analysis platform supporting DJI, Autel, Parrot, and others.
- [DroneLogbook](https://www.dronelogbook.com/) – Online flight logging and analysis service for UAV pilots and fleets.
- [Exmaps](https://www.exmaps.com/) – Flight tracking and analysis platform for drones.
- [DJI Assistant 2](https://www.dji.com/dji-assistant-2) – Official software for analyzing DJI drone logs.

## Flight Log Formats & Documentation

- [PX4 ULog Format](https://docs.px4.io/main/en/dev_log/ulog_file_format.html) – Documentation for PX4 ULog file structure.
- [ArduPilot Dataflash Logs](https://ardupilot.org/copter/docs/common-downloading-and-analyzing-data-logs-in-mission-planner.html) – Guide to ArduPilot log formats and analysis.
- [DJI Flight Log File Types](https://developer.dji.com/documentation/mobile-sdk/ios-tutorial/en/LoggingData.html) – DJI documentation on log files and telemetry data.
